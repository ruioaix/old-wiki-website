LAMP

在不同版本linux下lamp的安装

  • ubuntu10.04
    1. 在新立得--编辑--使用任务分组标记软件包--LAMP--安装过程中需要设置mysql的密码--12345
    2. 测试 http://localhost 看看是否ok。
  • ubuntu11.10
    1. 默认是没有安装synaptic的，可以在软件中心或者apt-get安装下。
    2. 安装的synaptic没有任务分组标记软件包，需要安装tasksel。

Ubuntu10.04下应用Apache2的userdir模块

  • userdir模块就是再/home/yourname/下建造一个用于web服务的目录。这样，在开发的时候，就可以不在/var/www/下了，就是可以不用每一次都sudo了，而且可以应用自己的vimrc等。
  • Ubuntu10.04下安装LAMP有一个简单的好用的方法：
    • 打开新立得软件包管理器。
    • 编辑
    • 使用任务分组标记软件包
    • 在其中选定LAMP，应用即可。
    • 中间要设置下mysql的root密码。
  • Ubuntu10.04下的Apache2的配置文件夹在/etc/apache2/下。
    • 其中httpd.conf是空文件，apache2.conf是全局的配置文件。
    • conf.d是附加的一些配置文件。
    • mods-available是可使用的模块，但不一定是 enabled 。
    • mods-enabled中都是上者的链接，都是 enabled 的。
    • 上两者中的文件都是一对一对使用的，一个load文件是模块本身，一个是conf文件。
  • Userdir模块的使用
    • 将mods-available中的文件链接到mods-enabled目录下。
      • ln -s /etc/apache2/mods-available/userdir.load /etc/apache2/mods-enabled/userdir.load
      • ln -s /etc/apache2/mods-available/userdir.conf /etc/apache2/mods-enabled/userdir.conf
      • 其实就是a2enmod userdir，就可以了。
    • 修改 userdir.conf 文件，将AllowOverride Filexxxxxx修改为AllowOverride All。
    • 修改 php5.conf 文件，按照里面的提示将其中对userdir模块的部分注释掉。如果不这样，在public_html中是不会解析php文件的。
  • 到这里就可以了，在/home/yourname/下建立public_html文件夹就可以了。

在Ubuntu10.04下设置phpmyadmin

  • 首先去官网下载最新的phpmyadmin。
  • 加压到usedir目录或者/var/www中。
  • cp config.sample.inc.php config.inc.php
  • 修改config.inc.php中的cookie的auth，添加任意字符串即可。
  • 完了（至少3.4版本是这样的）。

Ubuntu10.04下缺少php的一些扩展的时候

  • sudo apt-get install php5-mcrypt
  • sudo apt-get install php5-gd

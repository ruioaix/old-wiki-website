mysql

安装mysql

  • emerge -av mysql
  • emerge --config=dev-db/mysql-5.1.56
    • 如果 /etc/conf.d/hostname 中没有设置过，也就是hostname是localhost，那么，当前指令就会出错，修改之后，重新启动电脑就可以了，应该重新启动 /etc/init.d/hostname 也是可以的。
    • 在这一步中，会设置mysql的密码。
  • 测试mysql
    • /etc/init.d/mysql start
    • mysql -u root -h localhost -p
      • 这一步会要求密码，输入正确，就进入了mysql的环境。
      • SHOW DATABASES这个命令显示数据库当前情况。
      • exit

安装apache2

  • echo "www-servers/apache userdir">>/etc/portage/package.use
  • emerge -av apache
  • 在 /etc/conf.d/apache2 中的APACHE2_OPTS的最后，加入了-D USERDIR。
  • /etc/init.d/apache2 start，在启动的时候，发生了apr_sockaddr_info_get的错误，这个错误的解决方法是：在 /etc/hosts 文件中的127.0.0.1这一行在最后加上前面设置的hostname。

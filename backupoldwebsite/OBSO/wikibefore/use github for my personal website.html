用github做个人wiki

域名

  • 在[http://godaddy.com godaddy]购买一个域名，就如当下的[http://ray-x.me]。
    • 购买是一步，这一步网上搜索自己操作。
    • 第二步是设置域名，对于github来说，只需要设置
      • A(Host) 为207.97.227.245
      • CNAME(Alias) 中的 www 为yourname.github.com

github

  • 这里建立一个仓库yourname.github.com，用来放置html资源。
  • 其中需要一个 CNAME 文件，在其中写入ray-x.me即可。

评论系统

  • 使用disqus评论系统。
  • 其安装使用通用行，修改javascript的代码，放入vimwiki的模板中，就可以了。

git和vimwiki

  • yum install git
  • git config --global user.name "xxx.xxx"
  • git config --global user.email "xxx@xxx"
  • ssh-keygen -t rsa -C "xxx@xxx"
  • 复制rsa-pub,加入vimrc.
  • 下载vimwiki,:so %.
  • mkdir VimWiki xrfind.github.com
  • git clone git@github.com:xrfind/WikiSources.git VimWiki
  • git clone git@github.com:xrfind/xrfind.github.com.git xrfind.github.com
  • mkdir ~/.myscript
  • 修改.bashrc，加入export PATH="$PATH:~/.myscript"。
  • 加入vwikifirst,vwikilast。

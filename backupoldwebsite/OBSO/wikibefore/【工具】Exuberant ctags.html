Exuberant ctags

what is ctags

  • ctags 可以从你的或者第三方的源文件中提取出另外一个文件，这个文件包含了源文件中的各种有用信息，如标识符，函数原型等等。
  • 通过这个生成的文件（默认的名字是tags），我们可以快速的切换到某一个标识符定义的地儿，或者函数定义的地儿。
  • ctags生成的tags文件通常被一些vim插件使用。

下载源代码自己编译

  • 下载地址：[http://ctags.sourceforge.net Exuberant ctags]

安装就5条命令：

  • tar zvxf ctags-5.8.tar.gz
  • cd ctags-5.8
  • ./configure
  • make
  • make install 这条命令会要求root权限，使用sudo；如果没有root权限，把第三条改为”./configure -prefix=/home/yourname“就可以了

安装完成后，就可以将ctags-5.8目录和源文件删除了。

ubuntu10.04 安装

  • sudo apt-get install exuberant-ctags

使用

  • 对于c++文件，omnicppcomplete插件的文档中提供了一个选择：
    • ctags -R --c++-kinds=+p --fields=+iaS --extra=+q .
      • --c++-kinds=+p 为C++文件增加函数原型的标签
      • --fields=+iaS 在标签文件中加入继承信息’类成员的访问控制信息‘以及函数的指纹
      • --extra=+q 为标签增加类修饰符
  • 一个比较有用的使用方法是先对linux系统中头文件生成一个tags：
    • ctags -R -f ~/xxx/yourtagsname /usr/include
    • yourtagsname就是那个tags文件，将之包含入tags路径中（vimrc中设置）。

配置

  • 在.vimrc中关于ctags的设置部分要这样写：
    • set tags=tags
    • set tags+=./tags 将当前文件夹下的tags文件加入
    • set tags+=~/xxx/yourtagsname 将/home/yourname/program下的tags文件加入

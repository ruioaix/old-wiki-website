Git & github

github

这是一个网站：[https://github.com github]，以git为基础，可以将自己的工程托管在其上面也就是说本地需要安装git。

github提供的服务分为收费和不收费两种。

对于开源公开的项目，不需要收费。如果是私有的项目，可以缴纳一定的费用建立一个私有的工程。

据说github就是以收费来支持不收费的开源项目。

可以自由申请一个帐号，按照网站上的help，可以很轻松的上手。

本地PC安装git

对于ubuntu：

  • sudo apt-get install git-core git-gui git-doc

本地PC和github的通讯

使用ssh key来建立一个安全的通讯连接。 ubuntu刚安装好之后，在个人目录下 ~ 应该是没有 .ssh 目录的，有就备份下。 执行，以下步骤：

  • ssh-keygen -t rsa -c "xrfind@gmail.com" 本人的email，命令执行会有写问题，然后要你输入一个passphrase，其会在你开机第一次提交到github时被要求输入

完成命令后，在 .ssh 下会有一个"id_rsa.pub"文件，打开，copy里面的内容，复制到github上的”Account Settings""SSH Public Keys""Add another public key"中的key框，title不用管，最后会显示你的email。

输入：ssh git@github.com，可以验证是否成功。

最后，输入：

  • git config --global user.name "xrfind"
  • git config --global user.email "xrfind@gmail.com"

本人两台电脑同时工作于一个项目

非常简单。 假设上一小结中设置的是第一台电脑，下面设置第二台电脑。

  1. sudo apt-get install git-core git-gui git-doc
  2. ssh-keygen -trsa -c "xrfind@gmail.com"
  3. ssh git@github.com
  4. git config --global user.name "xrfind"
  5. git config --global user.email "xrfind@gmail.com"

就是完全一样的。

执行：git clone git@github.com:xrfind/yourworkdirname youlocalworkdirname

就可以把github上面的工程完全的克隆下来。

Git

本人两台电脑上的shell script。

第一台：

  • #!/bin/bash
  • cd ~/workdirname 进入工作目录
  • git add . 对工作树下所有文档生成快照
  • git commit -a -m "first pc" 将索引提交到仓库，标记是第一台pc的更改
  • git pull 将远端仓库和本地仓库进行合并
  • git push 提交合并后的版本到仓库
  • cd

第二台：

  • #!/bin/bash
  • cd ~/workdirname
  • git add .
  • git commit -a -m "second pc"
  • git pull
  • git push
  • cd

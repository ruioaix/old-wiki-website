web server

  • 这里指的就是apache2.2。
  • mod_userdir模块特别注意下，对我有用。
  • yum install httpd
  • systemctl start httpd.service
  • systemctl enable httpd.service
  • systemctl stop httpd.service
  • systemctl disable httpd.service
  • systemctl restart httpd.service
  • systemctl reload httpd.service这是重新加载配置文件。
  • service httpd graceful，在不影响访问的情况下，重新加载配置文件。
  • systemctl status httpd.service
  • systemctl is-active httpd.service

配置文件

  • /etc/httpd/conf/httpd.conf，这是主要的配置文件。
  • /etc/httpd/conf.d/，这目录下的文件被包含在了主要的配置文件中。
  • 在主要配置文件中的directory
    • <Directory "/var/www/html">
    • ...
    • </Directory>
    • 对特定的目录使用特殊的规则。
  • ifmodule
    • <IfModule mod_disk_cache.c>
    • ...
    • </IfModule>
    • 当一定的模块加载之后，使用规则。

网络

  • 浏览器输入的网址：<协定>://<主机位址或主机名称>[:port]/<目录资源>

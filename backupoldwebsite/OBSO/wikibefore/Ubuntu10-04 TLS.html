chromium+switchysharp+google app engine

  • 注册[http://code.google.com/appengine/ Google App Engine]帐号。
    • 使用google帐号登录Google app engine后，点击create an application创建一个应用程序。
    • 输入手机号码，接受创建验证码，必须由此过程，否则无法注册成功。中国用户使用+8612344445555这样的形式，瞬间短信来。
    • 输入验证码后就进入了创建应用程序的详细设置界面。填写application identifier和application title以及勾选同意服务条款，点save即完成创建。两者可以一样。
  • 安装Google app engine和fetchserver
    • 安装python，这个在你的终端试下命令，如果有，就不需要安装了，ubuntu10.04是自带的。
    • 下载[http://code.google.com/appengine/downloads.html google app engine sdk]，我下载的是python版本的，将sdk解压到任意目录。
    • 打开[http://code.google.com/p/gappproxy/downloads/list] ， 下载fetchserver，解压fetchserver到google app engine sdk目录。
    • 打开fetchserver文件夹内的app.yaml文件，修改第一行your_application_name为刚才输入的application identifier，就是前面注册时候的那个。
    • 上传fetchserver：打开终端，进入google app engine sdk目录，运行export PATH=$PATH:/home/ray/google_appengine/，这里要改成自己的目录。
    • 在终端继续运行命令appcfg.py update fetchserver，会要求你输入google的账户和密码，之后关闭结束上传。
    • 到这里，google app engine sdk这个文件夹就可以删掉了，没用了。
  • 安装并运行gappproxy
    • 下载[http://code.google.com/p/gappproxy/downloads/list] ，下载gappproxy。
    • 解压到任意文件。
    • 修改proxy.conf，在fetchserver一栏里面填写自己的appengine地址，并去掉这一行开头的#号，相邻的一行不用管。
    • 在终端里输入python proxy.py，如果看到http enabled：yes的字样，则代理已经开启了。
  • chromium安装switchysharp
    • 直接在扩展工具中search，然后安装。
    • 然后，在情景模式中设置一个，http代理为127.0.0.1，端口为8000，即可。
    • 点击地址栏右边的switchyproxy按钮，选择自动模式。
    • 如果刚才python proxy.py终止了，那么打开一个终端，输入，回车。
    • 在浏览器中打开facebook.com，不成功。
    • 点击地址栏右边的switchyproxy按钮，点击网址，选择新设置的情景模式，刷新页面应该就可以看到了。这是一次性的。
    • 如果希望长久保持可以访问facebook，那么就点击网址上面的新建规则，那么会保存下如下*://facebook.com/*这样的规则，星号是通配符。那么下次再访问符合通配规则的网址，就会自动的使用代理进行链接。
  • 设置代理后台开机启动。
    • 首先将gappproxy文件夹放入你要放的地方。
    • 编写一个shell脚本，内容是cd gappproxy，python proxy.py，编写完毕后，chmod，让其具有可执行特性。
    • 在系统-首选项-启动应用程序中添加一个启动项，启动项路径就是你shell脚本的路径，保存。
  • 至此，每次你开机的时候，代理服务都在后台运行，使用chromium的时候，switchyproxy最方便的是设置为自动模式，然后在使用中不断的添加规则。由于switchyproxy只是针对特定规则启用代理，所以，不是规则的网址，就不使用代理，非常方便惬意。
  • 而且，google的代理，访问google是嘎嘎的，也不会有词屏蔽。
  • 事实上，我搞这个，就是因为google东西时要死不活的样子实在让人吐血。

fcitx输入法

目前官网上还是3.x的版本，与4.x的没法比。 安装fcitx4的方法：

  1. sudo add-apt-repository ppa:wengxt/fcitx-nightly
  2. sudo apt-get update
  3. sudo apt-get install fcitx fcitx-config-gtk fcitx-sunpinyin
  4. im-switch -s fcitx-z default

前面三步是用来安装的，第四步是用来设置fcitx为默认输入法的，第四步不是必须的，可以在“系统”“系统管理”“语言支持”中做出对应修改。

需要重启，才可以生效。

  • 最后，最好删除掉 /etc/apt/source.list.d/ 下面相关的ppa。

3D效果

要关注的是：驱动和compiz。

  1. 驱动ubuntu10.04会自动识别显卡，建议安装，安装即可。
  2. compiz的安装
    • 在新立得里面搜索以下：
      • python-compizconfig
      • compizconfig-settings-manager
      • compiz-plugins
      • compiz-core
      • compiz-gnome
      • libemeraldengine0
      • emerald
      • libdecoration0
      • compiz-fusion-plugins-extra
      • fusion-icon
      • compizconfig-backend-gconf
  3. 安装完毕后在“应用程序”“系统工具”中就有compiz了，点击就可以进行设置。
  4. 开机自动启用3D桌面：在“系统”“首选项”“启动应用程序”里面“添加”，添加中的命令写fusion-icon，其他随便。

编写html

这个zip的包中，我只copy了ftplugin中html到.vim的ftplugin中。其他的browser和mangle我没有使用。

很多人使用zencoding这个插件，没用过，不做评论。

编写c++

使用了两个插件：

  1. [http://ctags.sourceforge.net Exuberant ctags]
  2. [http://www.vim.org/scripts/script.php?script_id=1520 OmniCppComplete]

ctags安装

安装就5条命令：

  • tar zvxf ctags-5.8.tar.gz
  • cd ctags-5.8
  • ./configure
  • make
  • make install 这条命令会要求root权限，使用sudo；如果没有root权限，把第三条改为”./configure -prefix=/home/yourname“就可以了

安装完成后，就可以将ctags-5.8目录和源文件删除了。

OmniCppComplete安装

安装就一条，之前将omnicppcomplete-0.41.zip放入.vim文件夹中，然后

  • unzip omnicppcomplete-0.41.zip

配置

ctags的使用就是在某个文件夹下使用ctags -R或者类似的命令，然后在该某个文件夹下会产生一个tags文件，这个文件就包含了标签。其他基于ctags的插件都使用这个标签。

在.vimrc中关于ctags的设置部分要这样写：

  • set tags=tags
  • set tags+=./tags 将当前文件夹下的tags文件加入
  • set tags+=~/program/tags 将/home/yourname/program下的tags文件加入

在我的.vimrc中，设置如下：

  • "--ctags setting----------------------------------------------------------------------------------------
  • set tags=tags
  • set tags+=./tags
  • set tags+=~/Geant4/G4InstallDir/geant4/geant4.9.2.p02/source/tags
  • set tags+=~/Geant4/G4InstallDir/CLHEP/include/tags
  • set tags+=/usr/include/c++/4.4/tags

其中4‘5行是Geant4相关，6行是c++的标准库。

我再上面的4’5‘6文件夹下都产生了tags，使用的命令是：ctags -R --c++-kinds=+p --fields=+iaS --extra=+q。

  • --c++-kinds=+p 为C++文件增加函数原型的标签
  • --fields=+iaS 在标签文件中加入继承信息’类成员的访问控制信息‘以及函数的指纹
  • --extra=+q 为标签增加类修饰符

对omnicppcomplete的设置如下：

  • "-- omnicppcomplete setting --
  • set completeopt=menu,menuone
  • let OmniCpp_MayCompleteDot = 1 autocomplete with .
  • let OmniCpp_MayCompleteArrow = 1 autocomplete with ->
  • let OmniCpp_MayCompleteScope = 1 autocomplete with ::
